# Yelp Restaurant Intelligence

**Advanced Machine Learning Final Project**

Dual deep-learning pipeline applied to the Yelp Open Dataset:

| Model | Task | Test Accuracy |
|-------|------|--------------|
| ResNet-18 (Transfer Learning) | Photo category classification | **94.7%** |
| BiLSTM | Review sentiment analysis | **72.0%** |

## How to use

1. Click **"🎲 Try a demo example"** to load a pre-built scenario, or
2. Upload your own restaurant photo and paste a review.

The app combines both predictions into a **business action** recommendation.
